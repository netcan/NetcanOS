◆ init_paging()

void init_paging ( )

初始化分页管理。

分配页框，使得物理地址和虚拟地址一致，并为已用过的内核空间分配页框。 由于内核加载到0x1000，后面有VRAM位于0xb8000，并且栈指针位于0x90000，所以覆盖完整个1MB空间。
